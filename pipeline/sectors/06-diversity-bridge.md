## Research Instructions

1. For NIH-funded programs (MARC, RISE, BRIDGES): these are awarded to universities, but find the NIH program portals
2. For scholarship organizations: focus on internship/research components, not just scholarships
3. Note stipend amounts — many diversity programs are fully funded
4. Note which programs require students to be at a specific institution vs. open nationally
5. LSAMP and MARC/RISE are at specific universities — describe the program type, link to NSF/NIH program page

## Output

Return ONLY a JSON array following the OUTPUT_SCHEMA. No explanation, no markdown — pure JSON.

Minimum 20 programs. Use verified URLs. Tag relevant programs with "diversity", "first-gen", "underrepresented".
